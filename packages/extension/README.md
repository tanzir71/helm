# Helm for Visual Studio Code

Helm is a beginner-friendly BYOK coding agent with streaming chat, approval-gated edits and
commands, queue/steer controls, workspace context, skills, checkpoints, and first-class profiles for
both frontier and open-weight models.

Open the Helm compass in the Activity Bar, choose **Set up a provider**, save your key, and test the
connection. Keys remain in VS Code SecretStorage. Start in Chat mode for read-only exploration or
Agent mode for native diff and command approvals.

The **Assist** tab keeps the normal direct workflow. The **Solo** tab automatically plans a task,
waits for one plan approval, promotes the task to the persistent goal, and executes the plan step by
step under Agent safety. Accepted edits reveal their real workspace file so progress stays visible
in the editor.

The extension has no account system, server, or telemetry.
